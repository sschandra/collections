Getting Started

Installing Node dependencies and Bower dependencies (Optional for project setup)

  1. Download or clone the master using repository clone url For Example clone to your file system use the flowing command in windows "cmd" prompt. Make sure you need to download and install the git package in your machine > git clone https://github.com/sschandra/collections.git
  2. Type following npm command to install the dependencies into your project. Make sure you require node installed in your machine. > cd collections
    > npm install
